# Contributing to Resume

This project helps job seekers create better resumes with AI assistance.

## Getting Started

```bash
# Clone the repo
git clone https://github.com/ianalloway/Resume.git
cd Resume

# Install dependencies
pip install -r requirements.txt

# Run the CLI
python main.py --help
```

## Project Structure

- `main.py` - Entry point for the CLI
- `career_agent.py` - AI agent for resume optimization
- `document_parser.py` - PDF parsing utilities
- `ai_providers.py` - Support for different AI providers

## Adding New Features

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Code Style

- Use Python 3.9+
- Follow PEP 8
- Add docstrings to new functions

## Issues

Found a bug? Open an issue on GitHub.
